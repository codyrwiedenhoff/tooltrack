# ToolTrack

## Project Overview
ToolTrack is an advanced tool management application designed to assist users in tracking and organizing their tools efficiently. Whether you are a professional contractor or a DIY enthusiast, ToolTrack keeps your tools organized and easily accessible.

## Tech Stack  
- **Frontend:** React  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Deployment:** Railway and Vercel

## Features
- User authentication  
- CRUD operations for tools  
- Search and filter tools  
- Responsive design  
- Integration with cloud storage

## Setup Instructions
1. Clone the repository:  
   ```bash  
   git clone https://github.com/codyrwiedenhoff/tooltrack.git  
   ```  
2. Navigate to the project directory:  
   ```bash  
   cd tooltrack  
   ```  
3. Install dependencies:  
   ```bash  
   npm install  
   ```  
4. Set up environment variables:  
   Create a `.env` file in the root directory and set your variables according to the `.env.example` file.
5. Start the development server:  
   ```bash  
   npm start  
   ```

## Deployment Guide for Railway
1. Sign in to Railway and create a new project.  
2. Connect your GitHub repository.  
3. Specify the build command and start command.  
4. Configure your environment variables in the Railway dashboard.  
5. Deploy your application.

## Deployment Guide for Vercel  
1. Sign in to Vercel and create a new project.  
2. Import your GitHub repository.  
3. Configure the build settings, if necessary.  
4. Deploy your application.

For more detailed instructions, refer to the official documentation for Railway and Vercel.
